## Tableau Dashboard – Campaign Performance Analysis
## Overview
This project presents an interactive Tableau dashboard focused on analyzing **marketing campaign performance** across multiple dimensions such as **location, channel, and time (quarterly trends)**.

The goal is to derive actionable insights into **profitability, cost efficiency, and engagement performance** across digital channels.

## Key Insights
**City-wise Profit Trends:** Cities like Bangalore and Mumbai consistently generate higher profits across quarters
**Channel Performance:** Instagram and Twitter show stronger profit contribution compared to Facebook in multiple regions
**Spend vs CPC Relationship:** Higher ad spend does not always translate into efficient cost-per-click, indicating optimization opportunities
**CTR Analysis:** Click-through rates vary significantly across channel-device combinations, highlighting targeting gaps

## Tools & Technologies
Tableau (Data Visualization & Dashboarding)

## Dashboard Features
Profit analysis segmented by **Date, City & Channel**
Scatter plot showing **Spend vs CPC relationship**
Combined visualization of **Channel & Device CTR performance**
Interactive filters for deeper data exploration
